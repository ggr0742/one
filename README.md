## Security Features

Every plan includes free SSL certificates through Let's Encrypt, so your site uses HTTPS. They auto-renew, so you don't have to remember to update them.

Daily automated backups save your files, databases, and emails. The basic plans give you backups, but only higher tiers let you restore them yourself. On cheaper plans, you contact support to restore a backup.
